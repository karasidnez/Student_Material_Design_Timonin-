# Student Material Design

Приложение для отображения списка студентов с использованием принципов Material Design 3 и Jetpack Compose.

## Цель работы

Изучение принципов Material Design в Android-приложениях с использованием Jetpack Compose. В ходе работы было создано приложение «Список студентов», демонстрирующее использование цветовых схем, типографики, форм и анимаций Material Design.

## Функциональность

- Отображение списка студентов с аватарками, именами и возрастом
- Раскрывающиеся карточки с подробным описанием каждого студента
- Плавная анимация при раскрытии/сворачивании карточек
- Поддержка светлой и тёмной темы оформления
- Адаптивный дизайн с использованием компонентов Material Design 3
- Верхняя панель приложения (TopAppBar) с логотипом и названием

## Технологии

- **Язык программирования:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Дизайн-система:** Material Design 3
- **Компоненты:**
  - Scaffold, Card, TopAppBar
  - LazyColumn для эффективной работы со списками
  - AnimatedContent, Spring Animation для анимаций
- **Шрифты:** Montserrat, Abril Fatface (Google Fonts)
- **Графика:** Векторные и растровые изображения
